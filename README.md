🎯 <strong>Project Objective</strong>
The goal of this project is to:
- Build a functional and visually clean login form
- Explore different layout techniques in CSS
- Strengthen understanding of alignment and centering

🛠️ <strong>Tech Stack</strong>
- HTML5 — Semantic structure
- CSS3 — Styling and layout


<strong>Key learning Focus</strong>
This project emphasizes:


- 📱 Responsive Design — Works across different screen sizes
- ♿ Accessibility — Semantic HTML & readable structure
- 📦 CSS Box Model — Spacing and layout control
- 🔄 Flexbox — Component alignment and distribution
- 🧩 CSS Grid — Layout structuring
- 📐 Positioning — Relative & absolute placement
- ✍️ Typography — Visual hierarchy and readability

⚡ <strong>Special Constraint</strong>
This project was built with a specific challenge:
Implement 3 different centering techniques within the layout.

🔍 Centering Methods Used
Flexbox Centering — using display: flex, justify-content, align-items
Grid Centering — using display: grid and place-items: center
Positioning + Transform — using position: absolute with transform: translate(-50%, -50%)

This helped deepen my understanding of how different CSS systems handle alignment and layout.

✨ <strong>Features</strong>
- Clean and modern login form UI
- Multiple centering techniques demonstrated
- Responsive and well-aligned layout
- Structured and readable code
